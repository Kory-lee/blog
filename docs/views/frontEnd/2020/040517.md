## 第一部分：数组

### 1. `all`: 布尔全等判断

```js
const all = (arr, fn = Boolean) => arr.every(fn);
all([4, 2, 3], (x) => x > 1); // true
all([1, 2, 3]); // true
```

### 2. `approximatelyEqual`约等于

```js
const approximatelyEqual = (v1, v2, epsilon = Number.EPSILON) => Math.abs(v1 - v2) < epsilon;
```
